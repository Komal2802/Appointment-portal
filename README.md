# Appointment Management Portal

A mini web application built using React.js for managing doctor appointments.

## Features

- Display list of doctors
- Book doctor appointments
- Edit appointments
- Delete appointments
- Form validation
- Smooth navigation to booking section
- Local storage support

## Technologies Used

- React.js
- CSS3
- Vite

## How to Run the Project

1. Clone the repository

```bash
git clone <repository-link>
```

2. Navigate to project folder

```bash
cd appointment-portal
```

3. Install dependencies

```bash
npm install
```

4. Start development server

```bash
npm run dev
```
